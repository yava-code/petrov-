# аналізатор дзвінків автосервісу

програма транскрибує аудіозаписи вхідних дзвінків автосервісу за допомогою faster-whisper, оцінює їх через швидке апі cerebras (модель llama-3.3-70b) за чеклистом та вивантажує детальний звіт в excel.

## як працює програма
- при запуску створює потрібні папки (`data/audio`, `data/transcripts`, `out`) та копіює шаблон таблиці
- бере mp3-файли з папки `data/audio`
- дату та номер телефону клієнта витягує з назви файлу (формат назви: `2025-09-10_15-52_0632838007_incoming.mp3`)
- транскрибує розмову та зберігає текст у `data/transcripts/` (якщо файл уже транскрибовано, бере готовий текст з кешу)
- аналізує розмову через cerebras за чеклистом та визначає тип робіт
- заповнює звіт у `out/Звіт_заповнений.xlsx` та фарбує коментар та оцінку розмов з поганим результатом у червоний колір

## швидкий запуск
1. встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```
2. створіть файл `.env` у корені проєкту та вкажіть ваш ключ для cerebras:
   ```env
   CEREBRAS_API_KEY=ваш_ключ
   ```
3. запустіть скрипт для створення структури папок:
   ```bash
   python main.py
   ```
4. закиньте аудіофайли `.mp3` у папку `data/audio` та запустіть `python main.py` повторно для аналізу.
