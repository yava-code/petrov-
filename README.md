# аналізатор дзвінків автосервісу

програма розпізнає аудіозаписи дзвінків за допомогою deepgram api та аналізує роботу менеджерів за чеклистом через cerebras api (модель zai-glm-4.7) з вивантаженням результатів у таблицю excel.

## режими роботи

програма може запускатися у локальному режимі або в режимі інтеграції з хмарою google drive.

при запуску без аргументів програма працює повністю локально з аудіофайлами в папці `data/audio` та не потребує налаштування google акаунту.

при запуску з прапорцем `python main.py --drive` активується хмарний режим. у цьому режимі програма автоматично завантажує нові записи зі спільної папки на диску, аналізує їх локально, а потім вивантажує mp3, текстові транскрипти та підсумковий звіт excel у вашу особисту робочу папку на google drive.

## інструкція для хмарного режиму (--drive)

для роботи з google drive використовується десктопний oauth потік. це дозволяє програмі отримати доступ до папок, які розшарені на ваш особистий акаунт.

спочатку створіть проєкт у google cloud console, увімкніть google drive api та створіть облікові дані типу oauth client id з типом додатку desktop app. завантажте отриманий файл конфігурації у форматі json, перейменуйте його в `credentials.json` та покладіть у корінь проєкту поряд з main.py.

також у файлі `.env` потрібно вказати id папки-джерела з дзвінками у змінній `SOURCE_FOLDER_ID`. за замовчуванням там уже встановлено наданий id.

при першому запуску з прапорцем `--drive` у вашому браузері автоматично відкриється сторінка авторизації google. увійдіть у свій особистий акаунт та надайте дозволи додатку. після цього програма автоматично збереже токен у файл `token.json` і при наступних запусках браузер відкриватися не буде.

## швидкий запуск

встановіть потрібні залежності:
```bash
pip install -r requirements.txt
```

створіть файл `.env` та вкажіть ключі та налаштування:
```env
CEREBRAS_API_KEY=ваш_ключ_cerebras
DEEPGRAM_API_KEY=ваш_ключ_deepgram
SOURCE_FOLDER_ID=1R3hDscEc_Ujh1FytqWROg4tS__qcO1Ub
```

запустіть локальний аналіз:
```bash
python main.py
```

або запустіть аналіз з інтеграцією google drive:
```bash
python main.py --drive
```
